# Analýza nejúspěšnějších umělců na YouTube
## Popis projektu
Tento projekt se zaměřuje na analýzu dat o nejsledovanějších YouTube kanálech. Cílem bylo procvičit si SQL dotazy, filtrování dat a řazení podle různých metrik.

## Dataset
Data obsahují informace o umělcích, jejich celkovém počtu zhlédnutí, průměrné sledovanosti na video a počtu videí, která překonala hranici 100 milionů zhlédnutí.

Zdroj: Kaggle

Formát: SQLite / CSV

## Ukázky SQL dotazů
V tomto projektu jsem řešil například:

Nejlepší průměrná sledovanost: Kdo má nejvlivnější videa.

Portfolio hitů: Kdo má nejvíce videí s více než 100M zhlédnutí.

Vyhledávání: Filtrování konkrétních interpretů pomocí LIKE.

## Použité technologie

SQL (SQLite/PostgreSQL)

DBeaver (Databázový manažer)
